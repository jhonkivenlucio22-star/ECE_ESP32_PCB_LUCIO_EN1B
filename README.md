## Project Description
This repository contains a set of simple Arduino projects designed to demonstrate fundamental concepts of digital and analog input/output, control structures, and basic circuit design. Each project is built to be beginner-friendly, covering core skills such as reading sensor values, controlling LEDs, using switches, and understanding the relationship between code and hardware. The projects are developed as part of coursework to apply programming and electronics principles in practical, hands-on implementations.

---

## Features

---

- Read digital signals from push buttons and switches

- Output digital HIGH/LOW signals to components like LEDs, buzzers, and relays

- Read analog values from sensors such as potentiometers, photoresistors, and temperature sensors

- Use control structures (if, else, while, for) to create decision-making and repeating behaviors

- Demonstrate proper pin configuration with pinMode()

- Show usage of standard C++ syntax and Arduino-specific functions

- Well-commented code for easy understanding and modification
